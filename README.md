# 🏨 Hotel Reservation System

![.NET](https://img.shields.io/badge/.NET-6.0-purple)
![C#](https://img.shields.io/badge/C%23-Programming-blue)
![Status](https://img.shields.io/badge/status-completed-brightgreen)

A console-based Hotel Room Booking System built using C#, LINQ, and clean architecture principles, designed to simulate real-world hotel reservation workflows.

---

# 📌 Overview

This project demonstrates a structured approach to building a hotel management system using:

- Object-Oriented Programming (OOP)
- LINQ queries
- Service-based architecture
- Interactive console UI

The system allows users to view, filter, and book hotel rooms efficiently.

---

# ✨ Features

- View all rooms  
- View available rooms only  
- Book rooms interactively  
- Prevent double booking  
- Search rooms by type  
- Get cheapest room  
- Count booked rooms  
- Dashboard statistics  
- Clean and colored console UI  

---

# ⚙️ Technologies Used

- C#

- Object-Oriented Programming (OOP)

---

# 🎮 How It Works

1. System loads predefined room data (Seed Data)  
2. User interacts with menu-driven console UI  
3. LINQ processes filtering and queries  
4. Booking updates room status in real-time  
5. Dashboard displays system statistics
