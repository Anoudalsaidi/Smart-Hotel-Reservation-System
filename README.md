# 🏨 Hotel Room Booking System - LINQ Project
---------------------------------------------

## 📌 Project Overview
This project is a simple C# Console Application that demonstrates the use of LINQ operations using a Hotel Room Booking System example. It works with in-memory data (List of Rooms) without a database.

---

## 🧠 Concepts Used
- LINQ (Where, Select, OrderBy, GroupBy, Count, Average, FirstOrDefault)
- C# Classes and Objects
- Collections (List)
- Console Application
- Data Filtering & Sorting

---

## 🏨 Room Structure
Each room contains:
- Id
- RoomNumber
- Type (Single / Double / Suite)
- PricePerNight
- IsBooked
- Floor

---

## ⚙️ Features
- Show available rooms
- Display room numbers only
- Find first available suite
- Sort rooms by price
- Count booked rooms
- Calculate average price
- Group rooms by type
- Filter rooms by floor and price

---


## 📸 Output Example
Data is displayed in a clean table format in the console.
